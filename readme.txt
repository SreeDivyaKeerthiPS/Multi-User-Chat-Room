First open terminal and go to path directory where the server client files are existing

Enter Server.py and then your Chat Server will be running


In another terminal go to the path directory where Server and Client files are existing

Enter Client.py and a Chat UI will pop up where you can chat with other clients. 
You can create multiple clinets my opening multiple command terminals. 
You can view the connected clients from the Server side as well

To exit Chat UI , you can press the quit button on the UI pop up
